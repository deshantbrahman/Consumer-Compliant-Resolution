# Consumer Complaint Resolution Using Machine Learning

## Overview
This project focuses on analyzing consumer complaint data and predicting customer dispute categories using Machine Learning algorithms. The project includes data preprocessing, feature engineering, text processing, visualization, and model building techniques.

## Features
- Consumer complaint data analysis
- Text preprocessing using NLP
- Feature engineering and data cleaning
- Multiple Machine Learning algorithms
- Data visualization and EDA
- Complaint category prediction
## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NLTK
- Scikit-learn
- SMOTE

## Machine Learning Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- AdaBoost
- Gradient Boosting
- KNN
- SVM
## Dataset
The dataset contains consumer complaint records including:
- Product
- Issue
- Company
- State
- Complaint Narrative
- Company Response
- Timely Response
- Consumer Dispute Status
## Project Workflow
- Data Cleaning
- Missing Value Handling
- Feature Engineering
- NLP Text Processing
- Data Visualization
- Model Training
- Model Evaluation
